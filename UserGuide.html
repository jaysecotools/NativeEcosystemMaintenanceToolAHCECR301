EcoMaintain User Guide

Overview

EcoMaintain is a comprehensive web-based application designed to help environmental professionals manage native ecosystem maintenance activities in compliance with AHCECR301 requirements. This tool provides functionality for site assessment, maintenance planning, species monitoring, compliance tracking, and reporting.

Key Features
  • Site assessment and ecosystem health monitoring
  • Maintenance activity planning and tracking
  • Species inventory and monitoring
  • Compliance tracking with environmental legislation
  • Comprehensive reporting capabilities
  • Interactive mapping and spatial data visualization
System Requirements
  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection (for external libraries and mapping)
  • JavaScript enabled
  • Recommended screen resolution: 1280x768 or higher

Getting Started

Initial Setup

1 Access the Application: Open the HTML file in your web browser

2 Review Default Data: The application comes with sample data to demonstrate functionality

3 Configure Site Settings: Navigate to the Settings tab to enter your specific site information

Tip: Take time to explore the sample data before entering your own information to understand how the application works.

Navigation

The application uses a tab-based navigation system with the following main sections:

  • Dashboard: Overview of key metrics and recent activities
  • Site Assessment: Record and review ecosystem health assessments
  • Maintenance Plan: Schedule and track maintenance activities
  • Species Register: Catalog and monitor flora and fauna
  • Monitoring: Record field observations and surveys
  • Reports: Generate compliance and activity reports
  • Legislation: Reference relevant environmental legislation
  • Settings: Configure application and user preferences

Dashboard

The dashboard provides an at-a-glance overview of your ecosystem management activities with key metrics and recent updates.

Key Components
  • Statistics Panel: Upcoming tasks, assessments, species records, and compliance percentage
  • Upcoming Tasks: List of scheduled activities with due dates
  • Site Health Indicators: Visual representation of ecosystem health metrics
  • Recent Observations: Latest monitoring records
  • Interactive Map: Geographic view of your site with zone overlays
  • Compliance Checklist: Quick view of compliance status with AHCECR301
Using the Dashboard

The dashboard is your starting point each time you use EcoMaintain. It helps you:

  • Quickly assess current priorities
  • Monitor compliance progress
  • Access recent activities and observations
  • Navigate to specific sections of the application
Tip: Use the export and print functions on the dashboard to create quick status reports for meetings or presentations.

Site Assessment

Conduct and record comprehensive site assessments to monitor ecosystem health and identify management priorities.

Creating a New Assessment

1 Click the "New Assessment" button on the Site Assessment tab

2 Enter assessment date and assessor name

3 Describe the site conditions and observations

4 Select ecosystem components present (flora, fauna, soil, water, etc.)

5 Rate overall ecosystem health using the standardized scale

6 Identify threats from the predefined list

7 Upload supporting photos if available

8 Save the assessment

Best Practice: Conduct assessments quarterly or after significant events. Use consistent methodology for comparable results over time.

Viewing Previous Assessments

All previous assessments are displayed in a table format with key information:

  • Browse the table to review past assessments
  • Click "View" to see detailed information for any assessment
  • Export assessments as PDF reports for documentation
  • Compare assessments over time to track trends

Maintenance Plan

Schedule, track, and manage all maintenance activities for your ecosystem management program.

Creating Maintenance Activities

1 Click "New Activity" button

2 Select activity type (weed control, revegetation, erosion control, etc.)

3 Set start and end dates

4 Specify location and assign personnel

5 Add detailed descriptions and any special notes

6 Set priority level (low, medium, high)

7 Set initial status (planned, in-progress, etc.)

8 Save the activity

Tracking Activities

Activities can be viewed in three different tabs:

  • Upcoming: Planned and in-progress activities
  • Completed: Finished activities with completion details
  • Overdue: Activities that are past their scheduled date

Calendar Integration

The Maintenance Calendar provides a visual overview of all scheduled activities:

  • View all activities on an interactive calendar
  • Click on events to see details
  • Different activity types are color-coded for easy identification
  • Switch between month, week, and day views
Tip: Use the calendar view to identify scheduling conflicts and optimize resource allocation across different activities.

Species Register

Maintain a comprehensive inventory of all flora and fauna species present in your management area.

Adding Species Records

1 Click "New Species" button

2 Enter scientific name (required) and common name

3 Classify by type (flora, fauna, fungi, other)

4 Set conservation status (native, introduced, weed, endangered, etc.)

5 Add descriptions and habitat information

6 Record last observation date

7 Upload images for identification

8 Save the species record

Managing Species Data

Species information can be viewed in two formats:

  • Card View: Visual presentation of key species with images
  • Table View: Comprehensive list with filtering capabilities

Additional functionality includes:

  • Export species data for reporting or external analysis
  • Link species to monitoring observations
  • Filter by type or conservation status
  • Quick actions for common tasks (control weeds, monitor threatened species)
Best Practice: Use standardized survey methods for consistency. Record both presence and abundance where applicable. Update species records after each survey.

Monitoring

Record field observations, surveys, and monitoring activities to track changes in your ecosystem over time.

Logging Monitoring Activities

1 Create new monitoring records for different types of observations

2 Record location, date, observer, and detailed findings

3 Associate observations with relevant species from your register

4 Upload photographic evidence when available

5 Save the monitoring record

Specialized Monitoring Forms

EcoMaintain includes specialized forms for different types of monitoring:

Fauna Sightings
  • Record specific animal observations
  • Note behavior and population counts
  • Link to species in your register
Vegetation Surveys
  • Document plant cover percentages
  • Record health ratings
  • Track changes in vegetation composition

Viewing Monitoring History

All monitoring activities are displayed in a comprehensive table:

  • Browse all monitoring records
  • Filter by type, date, or location
  • Generate specialized reports from monitoring data
  • Track trends over time

Access important regulatory information and track compliance with environmental legislation.

Legislative Reference

EcoMaintain provides information on key environmental legislation:

  • EPBC Act 1999: Federal environmental protection requirements
  • State/Territory Legislation: Jurisdiction-specific regulations
  • Local Government Regulations: Council-level requirements

Compliance Checklist

The interactive compliance checklist helps you track regulatory requirements:

  • Monitor status of each compliance item
  • Record evidence for compliance activities
  • Add notes and documentation references
  • Track overall compliance percentage
Best Practice: Regularly review legislative requirements. Maintain evidence for compliance activities. Document any non-compliance and remediation plans.

Settings

Configure the application to your specific needs and manage user preferences.

User Profile

Update your personal information and application preferences:

  • Name, email address, and contact information
  • User role (administrator, ecologist, field technician, volunteer)
  • Password changes for security

Site Configuration

Set up information specific to your management area:

  • Site name, location, and geographic details

Data Management

Tools for managing your application data:

  • Export all data for backup purposes
  • Create database backups
  • Reset test data (clears sample information)
Warning: The "Reset Test Data" function will permanently delete all sample data. Use with caution and ensure you have backups if needed.

Data Management

Understand how EcoMaintain handles your data and implement effective data management practices.

Storage

  • All data is stored locally in your browser using localStorage
  • No server connection required for basic functionality
  • Data persists between browser sessions on the same device

Backup and Export

To prevent data loss, regularly:

  • Export all data using the "Export All Data" feature
  • Create comprehensive backups before making significant changes
  • Individual sections (like Species) can be exported separately

Data Integrity

EcoMaintain includes features to maintain data quality:

  • Form validation ensures required information is complete
  • Required fields are clearly marked with asterisks (*)
  • Date fields use standard formats for consistency
  • Data relationships are maintained (e.g., species linked to observations)
Tip: For multi-user environments or enhanced data security, consider implementing a server-based version with proper authentication and database storage.

Best Practices

Implement these recommended practices to maximize the effectiveness of your ecosystem management program.

For Site Assessments

  • Conduct assessments quarterly or after significant events
  • Use consistent methodology for comparable results
  • Document threats with specific locations and severity
  • Include photographic evidence when possible

For Maintenance Planning

  • Schedule activities based on seasonal requirements
  • Prioritize high-impact tasks (weed control, erosion management)
  • Assign tasks to qualified personnel
  • Record completion details and any variations from plan

For Species Monitoring

  • Use standardized survey methods for consistency
  • Record both presence and abundance where applicable
  • Note behavioral observations for fauna
  • Update species records after each survey

For Compliance

  • Regularly review legislative requirements
  • Maintain evidence for compliance activities
  • Generate reports ahead of regulatory deadlines
  • Document any non-compliance and remediation plans

Troubleshooting

Solutions for common issues you might encounter while using EcoMaintain.

Common Issues and Solutions

Data Not Saving
  • Check that all required fields are completed
  • Ensure browser has not disabled localStorage
  • Try refreshing the page and re-entering data
Map Not Loading
  • Verify internet connection is active
  • Check if mapping services are blocked by firewall
  • Try refreshing the page
Calendar Not Displaying Events
  • Ensure activities have valid dates
  • Check that activities are not filtered out
  • Refresh the calendar view

Performance Tips

  • Limit the number of photos uploaded to maintain performance
  • Regularly export and clear old data if storage becomes limited
  • Use the table filtering and sorting features for large datasets
Support: For technical issues or enhancement requests, please contact your system administrator. Regular updates to the application may add new features or improve existing functionality.

Compliance with AHCECR301

EcoMaintain is designed to assist with compliance against the AHCECR301 unit of competency, which covers:

  • Implementing ecosystem maintenance procedures
  • Monitoring ecosystem health
  • Managing threats to native ecosystems
  • Maintaining compliance with legislative requirements

The application provides specific features to address each of these areas, with documentation and reporting capabilities to demonstrate compliance.
